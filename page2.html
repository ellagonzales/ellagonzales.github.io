Projects


I’ve had the chance to work on several projects that showcase my passion for tech and social good:

  • Whisker: A pet adoption app that connects users with pets based on preferences. Built using the RescueGroups API, Whisker features a swipe-to-save interface, and its intuitive design earned it “Best UI/UX Design.”
  • PORCH Mobile App: As the Product Manager, I’m leading the development of a mobile app for PORCH, a hunger relief non-profit. This app simplifies the food donation process for the community, with input from cross-functional teams and regular stakeholder collaboration.
  • HealthEats Analyst: I led a cross-functional team to create a model that uses NLP to assess restaurant healthiness. This project required translating technical concepts for a diverse audience and won “Most Compelling Team Presentation.”
  • UpCycle Creations: A sustainable fashion venture where I design and sell upcycled clothing, generating over $1,000 in net profit. This project allows me to combine creativity with sustainability, promoting conscious consumerism.

For more projects and code samples, check out my GitHub.
